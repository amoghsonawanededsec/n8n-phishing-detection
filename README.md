# Automated Phishing Detection Using n8n

## Overview

This project implements an automated email security workflow using n8n. The workflow monitors incoming Gmail messages, analyzes email content for phishing indicators, assigns a risk score, and sends alerts when suspicious emails are detected.

## Features

* Gmail Trigger Integration
* Email Content Extraction
* Keyword-Based Phishing Detection
* Risk Score Calculation
* Severity Classification
* Automated Gmail Alerts
* No-Code/Low-Code Security Automation

## Workflow

Gmail Trigger → Extract Email Data → Phishing Detection → Risk Classification → IF Condition → Gmail Alert

## Technologies Used

* n8n
* Gmail API
* JavaScript
* OAuth2
* Cybersecurity Automation
